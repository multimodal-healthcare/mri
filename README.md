#  MRI
This is the repository for the Magnetic Resonance Imaging (MRI) modality of the pipeline. Together with other modality, we build a multi-modal AI model to diagnose Cardiovascular Disease (CVD). 

## Requirements
`pydicom==2.3.0`  
`numpy==1.23.2`  
`pandas==1.4.4`  
`ipywidgets==8.0.2`  
`skimage==0.19.3`
